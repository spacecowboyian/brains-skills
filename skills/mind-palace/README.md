# Mind Palace

> Store and retrieve knowledge spatially by associating information with vivid locations in imagined or remembered spaces.

## Install

Paste into your AI chat:

```
Read https://raw.githubusercontent.com/spacecowboyian/brains-skills/main/skills/mind-palace/skill.md
then set up a new project in my Brains knowledge base using this skill.
```

## What this skill does

Sets up a Mind Palace knowledge base with a spatial folder structure and an AI agent that helps you encode and retrieve knowledge using the Method of Loci. Once you have a palace, the agent can:

- Help you choose and build a palace from a familiar space
- Place new memories at specific loci with vivid sensory anchors
- Walk you through the palace room by room for retrieval practice
- Retrieve any stored memory by navigating palace → room → locus
- Redesign faded anchors to make them more vivid and durable

## Example queries

- `I want to memorize the 12 cranial nerves — build me a palace`
- `Walk me through my childhood home palace`
- `What's stored in the kitchen?`
- `Place this new fact: the mitochondria produces ATP`
- `The anchor for locus 3 isn't sticking — redesign it`
