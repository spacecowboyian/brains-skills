# Johnny.Decimal

> Assign every piece of knowledge a unique numeric ID within a strict two-level hierarchy so anything can be found in seconds.

## Install

Paste into your AI chat:

```
Read https://raw.githubusercontent.com/spacecowboyian/brains-skills/main/skills/hippocampus/johnny-decimal/skill.md
then set up a new project in my Brains knowledge base using this skill.
```

## What this skill does

Sets up a Johnny.Decimal system with a strict numeric hierarchy and an AI agent that assigns IDs, maintains the index, and enforces the rules. Once your system has areas and categories, the agent can:

- Design your area structure from scratch based on your actual life domains
- Assign the next available ID for any new piece of content
- Maintain a master index of all areas and categories
- Navigate directly to any item by ID — no search needed
- Enforce the rules: max 10 areas, max 10 categories per area, no deeper nesting

## Example queries

- `Help me design my areas from scratch`
- `What ID should my new car insurance document get?`
- `What's in area 30?`
- `Show me the full system map`
- `I need to file notes from a client call — where does it go?`
