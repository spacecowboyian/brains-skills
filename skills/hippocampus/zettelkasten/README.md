# Zettelkasten

> Atomic, linked notes where each idea gets its own card and connections between cards surface emergent insight.

## Install

Paste into your AI chat:

```
Read https://raw.githubusercontent.com/spacecowboyian/brains-skills/main/skills/hippocampus/zettelkasten/skill.md
then set up a new project in my Brains knowledge base using this skill.
```

## What this skill does

Sets up a Zettelkasten with a notes folder, three note types, and an AI agent configured to help you capture, link, and synthesize ideas. Once you have notes, the agent can:

- Enforce one idea per note — and split when you write two
- Generate unique timestamp IDs for each zettel
- Discover connections between notes and explain why they matter
- Build Maps of Content (MOCs) that organize a theme into a reading order
- Process fleeting notes before they go stale
- Find orphan notes and suggest where they belong in the network

## Example queries

- `I just had an idea about how constraints drive creativity — capture it`
- `What do I have on motivation?`
- `What connects to note 202605291430?`
- `Build a map of content around habit formation`
- `Process my fleeting notes from this week`
- `What are my most connected ideas?`
