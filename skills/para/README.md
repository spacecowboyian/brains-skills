# PARA

> Organize all knowledge into four actionability-based buckets: Projects, Areas, Resources, Archives.

## Install

Paste into your AI chat:

```
Read https://raw.githubusercontent.com/spacecowboyian/brains-skills/main/skills/para/skill.md
then set up a new project in my Brains knowledge base using this skill.
```

## What this skill does

Sets up a PARA knowledge base with four folder buckets and an AI agent configured to triage, promote, and archive your content. Once you have pages, the agent can:

- Triage new captures into the right bucket with a single question
- Surface stale projects and prompt you to act or archive
- Promote resources into projects when they become actionable
- Search across all four buckets and report by bucket
- Walk you through archiving completed work and extracting reusable knowledge

## Example queries

- `I just finished reading a book on sleep — where should my notes go?`
- `What projects are active right now?`
- `What's gone stale in the last two weeks?`
- `Archive the podcast launch project and save anything reusable`
- `What do I have on nutrition?`
